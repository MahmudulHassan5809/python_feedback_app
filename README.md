# Python Feedback App

> Python Flask Feedback app that sends data to Postgres database and emails user

## Quick Start

```bash
# Add your DATABASE URI in app.py and your mail params in feedbackApp __init__.py
```

# DB Migration
  ```
  $ flask db init
  $ flask db migrate
  $ flask db upgrade
  ```

# Install Dependencies
```
pip install -r requirements.txt
```

# Serve on localhost:5000
python app.py
```
